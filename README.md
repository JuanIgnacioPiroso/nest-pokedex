# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejercutar
```
npm install
```
3. Tener Nest CLI instalado
```
npm i -g @nestjs/cli
```
4. Levantar la base de datos
```
docker-compose up -d
```
# Run in Development

1. Clone the repository  
2. Run:

```bash
npm install
```

3. Make sure Nest CLI is installed:

```bash
npm i -g @nestjs/cli
```

4. Start the database:

```bash
docker-compose up -d
```
